```js
	  <ButtonGroup>
		<Button secondary>Button</Button>
		<Button secondary>Button</Button>

	  </ButtonGroup>

```
ButtonGroups can be made full-width with the 'justify' property:
```js
	  <ButtonGroup justify>
		<Button secondary>Button</Button>
		<Button secondary>Button</Button>
		<Button secondary>Button</Button>
	  </ButtonGroup>
```
